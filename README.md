# Proyecto MERN - Programación Web 2

Aplicación web desarrollada utilizando el stack MERN:

- MongoDB
- Express
- React
- Node.js

## Descripción

Este proyecto consiste en una aplicación web con autenticación mediante JWT (JSON Web Token), separando el backend (servidor) y el frontend (cliente) en carpetas independientes.

## Instalación y Ejecución

### 1. Clonar el repositorio
```bash
git clone https://github.com/tu-usuario/proyecto-mern.git
cd proyecto-mern
```

### 2. Backend
```bash
cd backend
npm install
node server.js
```

El servidor se ejecutará en: `http://localhost:3000`

### 3. Frontend

Abre el archivo `frontend/index.html` directamente en tu navegador.

## Tecnologías Utilizadas
